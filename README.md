## MINECRAFT SERVER

- [https://hub.docker.com/r/itzg/minecraft-server](https://hub.docker.com/r/itzg/minecraft-server)
- [https://docker-minecraft-server.readthedocs.io/en/latest/](https://docker-minecraft-server.readthedocs.io/en/latest/)

**Quickstart**
```bash
cp example.env .env
```

```bash
mkdir mc-data
```

**UP(Run Your Server)**
```bash
docker compose up -d
```

**DOWN**
```bash
docker compose down
```
